# missionary-and-cannibals-pygame
A casual, interactive river-crossing puzzle game built in Python with Pygame. Safely guide 3 Missionary and 3 Cannibals across the river without letting the saints get outnumbered and eaten! Includes win/loss screens and restart mechanics.

##The Puzzle Rules
- The boat can hold a **maximum of 2 characters** at a time.
- The boat **cannot cross empty**; it needs at least one passenger to steer it.
- **Safety Rule:** If monsters ever outnumber saints on either bank of the river, the monsters will overwhelm the saints. (Note: Characters sitting in the boat count toward the total on whatever side the boat is currently parked!)

## Controls
* `S` - Load a Saint onto the boat
* `M` - Load a Monster onto the boat
* `X` - Unload the last character who boarded back onto the current bank
* `SPACE` - Sail the boat across the river
* `R` - Restart the game (available on the Game Over / Win screens)
* `ESC` - Quit the game immediately

## Prerequisites & Installation
Make sure you have Python installed on your system. You will also need the `pygame` library.
